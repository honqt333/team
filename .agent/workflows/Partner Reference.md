# 📘 وثيقة النظام الرسمية (Agent & Partner Reference)

---

## 1️⃣ تعريف النظام (Product Overview)

المنصّة هي نظام SaaS لإدارة وتشغيل مراكز الخدمات، مبني على مبدأ:

> **Core ثابت لا ينكسر + Modules مرنة قابلة للتفعيل**

تخدم المراكز من:
- أول تواصل مع العميل
- إلى الحجز
- إلى أمر العمل
- إلى الفاتورة
- إلى الدفع
- مع تتبع الأداء والنمو

---

## 2️⃣ الفئة المستهدفة (Target Market)

- ورش سيارات
- مراكز صيانة
- مراكز تلميع
- شبكات وسلاسل مراكز
- مراكز B2B (أساطيل / شركات)

---

## 3️⃣ البنية المعمارية (Architecture Concept)

### 🧱 Core (أساسيات النظام)

> لا تُزال، ولا تُباع كإضافات

| المكون | الوصف |
|--------|-------|
| Multi-Tenant | مستأجر واحد = علامة واحدة |
| الفروع | فروع متعددة داخل المستأجر |
| المستخدمون | أدوار + صلاحيات دقيقة |
| الاشتراكات | تجربة + انتهاء (Read-only) |
| العملاء | بيانات العملاء الكاملة |
| المركبات | مع الماركات والموديلات |
| الخدمات | مع الأقسام والأسعار |
| أوامر العمل | الدورة الكاملة |
| عروض الأسعار | مع التحويل لأمر عمل |
| الفواتير | حالات الدفع |
| المخزون الأساسي | منتجات + صرف + جرد |
| التقنيين | لوحة الفني (بدون أسعار) |
| Inbox موحّد | الإشعارات الداخلية |
| Dashboard | إحصائيات أساسية |
| صفحة المستأجر | صفحة عامة واحدة |
| Audit Log | سجل العمليات |
| Backup | نسخ احتياطي يومي |

## 6️⃣ صفحة المستأجر (Public Tenant Page)

صفحة واحدة لكل مستأجر تعرض:
- نبذة عن المركز
- الخدمات المتاحة
- معاينة الأسعار
- الفروع والمواقع

وتُمكّن:
- الحجز المباشر
- طلب عرض سعر
- التواصل

> ❌ لا صفحات فروع منفصلة
> ❌ لا مواقع مستقلة
> ✔ تحكم مركزي كامل

---

## 7️⃣ رحلة العميل (Customer Journey)

```
العميل → يبحث عن خدمة
      → يفتح صفحة المركز
      → يطلب عرض سعر أو يحجز
      → يتلقى تأكيد
      → يأتي للمركز
      → يتم فتح أمر عمل
      → يتم التنفيذ
      → يستلم فاتورة
      → يدفع
      → (اختياري) يحصل على نقاط ولاء
```

- عدد السيارات المخدومة يومياً

## 🔟 التنبيهات الذكية

| الحدث | التنبيه |
|-------|---------|
| أمر عمل متأخر 24 ساعة | للمدير + الفني |
| عرض سعر لم يُرد عليه 3 أيام | للموظف |
| اشتراك ينتهي قريباً | للمستأجر |
| فاتورة غير مدفوعة 7 أيام | للعميل + المدير |
| نفاد كمية منتج | للمخزن |
