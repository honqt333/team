# خريطة طريق المهام المتبقية للنظام (Remaining Tasks)

تمثل هذه الوثيقة المهام المتبقية لاكتمال نظام Carag (Khidma Pro) بشقيه: التشغيلي (للمراكز) والإداري (للمالك).

## 🛠️ أولاً: المهام التشغيلية (للمراكز - الأولوية القصوى)

هذه المهام تكمل الدورة التشغيلية اليومية داخل المركز، وبدونها لا يمكن للمركز الاعتماد على النظام بشكل كامل.

- [ ] **لوحة الفنيين (Technician Portal - `app/my`)**:
  - [ ] واجهة خاصة للمهام (My Tasks) تعرض أوامر العمل المسندة للفني.
  - [ ] تصميم مخصص للجوال/التابلت لتسهيل الاستخدام داخل الورشة.
  - [ ] إخفاء تام للأسعار والتفاصيل المالية.
  - [ ] إمكانية تحديث حالة أمر العمل (قيد التنفيذ -> فحص -> مكتمل).
  - [ ] رفع الصور وإضافة ملاحظات الفحص الفني مباشرة من الجوال.

- [ ] **نظام الحجوزات والمواعيد (Bookings)**:
  - [ ] شاشة لجدولة المواعيد (Calendar View).
  - [ ] تحويل الموعد إلى أمر عمل بضغطة زر.

- [ ] **التنبيهات الذكية (Smart Alerts)**:
  - [ ] إشعار للمدير بأوامر العمل المتأخرة (> 24 ساعة).
  - [ ] إشعار بعروض الأسعار المعلقة (> 3 أيام).

---

## 🌍 ثانياً: الواجهة العامة للعملاء (Public Pages)

- [ ] **صفحة هبوط لكل مركز (Public Tenant Page)**:
  - [ ] صفحة عامة تعرض اسم المركز، الشعار، وموقعه.
  - [ ] عرض الخدمات المتاحة للجمهور.
  - [ ] بوابة مبسطة للعميل لطلب حجز موعد أو طلب عرض سعر (يصل مباشرة إلى لوحة تحكم المركز).

---

## 🎛️ ثالثاً: لوحة المالك ونظام الاشتراكات (System Admin - `/system`)

هذه المهام ضرورية لتحويل النظام إلى منتج SaaS قابل للبيع وتحصيل الاشتراكات.

- [ ] **إدارة الاشتراكات والدفع (Billing & Subscriptions)**:
  - [ ] ربط بوابة دفع (Moyasar / Tap) لدفع اشتراكات الباقات.
  - [ ] نظام التجديد التلقائي والتذكير بانتهاء الاشتراك.
  - [ ] إدارة الفترات التجريبية (Free Trials) وإيقاف الحسابات المنتهية.

- [ ] **متجر الإضافات (Modules Store)**:
  - [ ] شاشة تتيح للمالك بيع خدمات إضافية للمراكز (Add-ons).
  - [ ] بيع باقات رسائل SMS و WhatsApp.
  - [ ] تفعيل ميزات متقدمة (مثل تفعيل الدفع بالتقسيط تابي/تمارا).

- [ ] **إحصائيات المالك (Super Admin Dashboard)**:
  - [ ] إحصائيات الإيرادات الشهرية من الاشتراكات.
  - [ ] تقارير عن نمو المراكز وعدد المستخدمين.

---

## 🚀 رابعاً: التكاملات الخارجية (Integrations)

- [ ] **تكاملات الدفع والتقسيط**:
  - [ ] ربط بوابات دفع لتمكين المراكز من إرسال "روابط دفع" لعملائهم لدفع الفواتير أونلاين.
  - [ ] ربط BNPL (تابي / تمارا).

- [ ] **تكاملات المراسلة**:
  - [ ] WhatsApp Business API لإرسال فواتير وعروض أسعار للعملاء تلقائياً.
  - [ ] خدمات SMS (Unifonic / Taqnyat).
