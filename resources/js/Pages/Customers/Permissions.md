# 🔐 توثيق الصلاحيات - وحدة العملاء (Customers Module Permissions)

هذا الملف يوضح كافة مفاتيح الصلاحيات المستخدمة في صفحات العملاء لضمان التحكم الدقيق في الوصول.

## ١. صفحة قائمة العملاء (Customers Index)
**المسار**: `app/customers`

| المفتاح | الوصف | الإجراء المرتبط |
|---------|-------|----------------|
| `crm.customers.view` | عرض القائمة | الدخول لصفحة العملاء |
| `crm.customers.create` | إضافة عميل | ظهور زر "إضافة عميل" |
| `crm.customers.export` | تصدير البيانات | ظهور زر التصدير (Excel) |
| `crm.customers.import` | استيراد البيانات | ظهور زر الاستيراد |
| `crm.customers.print` | طباعة القائمة | ظهور زر الطباعة |

## ٢. صفحة ملف العميل (Customer Show)
**المسار**: `app/customers/{id}`

| المفتاح | الوصف | الإجراء المرتبط |
|---------|-------|----------------|
| `crm.customers.update` | تعديل بيانات العميل | زر التعديل وزر الدمج |
| `crm.customers.delete` | حذف العميل | زر الحذف (مع التحقق من القيود) |
| `crm.vehicles.view` | عرض المركبات | رؤية تبويب المركبات وقائمة مركبات العميل |
| `crm.vehicles.create` | إضافة مركبة | زر "إضافة مركبة" داخل ملف العميل |
| `crm.work_orders.view` | عرض كروت العمل | رؤية تبويب كروت العمل المرتبطة |
| `quotes.view` | عرض التقييمات | رؤية تبويب عروض الأسعار المرتبطة |
| `invoices.view` | عرض الفواتير | رؤية تبويب الفواتير |

---

> 💡 **ملاحظة**: كافة العمليات الحساسة تسمح بالوصول التلقائي لمدير النظام عبر دالة `isAnyAdmin()` حتى في حال عدم وجود الصلاحية المباشرة.
